<h1 align="center">Full-Stack Todo App</h1>
<p align="center"><strong>A production-ready full-stack task management application with REST API and modern frontend.</strong></p>

---

## Features

- **RESTful API** — Clean backend with CRUD operations
- **Modern Frontend** — Responsive UI for task management
- **Docker Ready** — Containerized for easy deployment
- **Persistent Storage** — Data persists across sessions
- **Production Config** — Environment-based configuration

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Backend | Python (Flask) |
| Frontend | HTML/CSS/JS |
| Deployment | Docker, Vercel |
| API | REST |

---

## Quick Start

```bash
git clone https://github.com/Ub207/full-stack-todo-app.git
cd full-stack-todo-app
pip install -r requirements.txt
python app.py
```

### With Docker
```bash
docker build -t todo-app .
docker run -p 5000:5000 todo-app
```
